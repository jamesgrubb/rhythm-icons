Icon Library Add-in

Testing Instructions & Feedback Guide

Thank you for helping test the Icon Library add-in! This add-in provides quick access to your organisation's curated icon library directly within Microsoft Word and PowerPoint.

Estimated time: 10–15 minutes
1

📥 Install the Add-in

Desktop (Word / PowerPoint)

  1. Download the manifest.xml file from the link provided
  2. Open Microsoft Word or Microsoft PowerPoint
  3. Click the Insert tab → Add-ins → Get Add-ins
  4. In the dialog, click My Add-ins (left sidebar, bottom)
  5. Click Upload My Add-in (top-right), then Browse
  6. Select the manifest.xml file and click Upload

Web (Office.com)

  1. Go to office.com and sign in
  2. Open Word or PowerPoint (new blank document)
  3. Click Insert → Add-ins → More Add-ins
  4. Click Upload My Add-in, browse to select manifest.xml, and upload
2

🚀 Open the Add-in

  1. In the Home tab of the ribbon, find the Icon Library button
  2. Click Show Taskpane
  3. A side panel will open — you'll see a loading screen, then a sign-in screen
3

🔐 Sign In with Microsoft

  1. Click "Sign in with Microsoft"
  2. A popup window will appear — allow popups if prompted
  3. Sign in with your work/school or personal Microsoft account
  4. First-time users: you'll see a consent screen requesting "Read icons" permission — click Accept
  5. The popup closes and you should see a grid of icons

Troubleshooting: If the popup is blocked, allow popups for this site. If sign-in fails, open the browser console (F12) for error details. Microsoft Edge works best.

4

🎨 Test Icon Features

Browse Icons

Scroll through the grid and click category tabs at the top

Search Icons

Try searching "heart", "star", "user" — clear with the X button

Size Adjustment

Click S, M, or L to preview different icon sizes

Theme Colours

Try colour buttons: BG1, BG2, T1, T2, A1–A6 (A1 is default)

Background Options

Toggle "Circle background" and "Fill icon" — preview updates live

Insert an Icon

Click any icon — it inserts at your cursor. Look for the green "Icon inserted" toast
5

✅ Test Session Persistence

  1. Close the Icon Library taskpane (click X on the panel)
  2. Reopen it: Home tab → Icon Library → Show Taskpane
  3. Expected: Icons load immediately — no sign-in prompt

Please report: If you see the sign-in screen again after reopening, session restoration isn't working — that's an important bug to flag!

6

🚪 Test Sign Out

  1. Click the sign-out button (top-right corner — exit/door icon)
  2. A popup will appear to complete the sign-out
  3. Expected: You return to the sign-in screen
  4. Sign back in to confirm it works end-to-end

📝 What to Report Back

✅ What Worked

  • Sign-in was smooth
  • Icons loaded quickly
  • Inserted icon looks perfect
  • Search works great

❌ What Didn't Work

  • Sign-in errors (screenshot the error)
  • Icons didn't load (how long did you wait?)
  • Icon insertion failed
  • Performance/layout issues

💡 Suggestions Welcome

Missing features, UI/UX improvements, icon requests (categories or specific icons), or documentation feedback — all appreciated!

🐛 When Reporting an Issue, Please Include

  1. Browser/Office version — e.g. "Word Desktop 2021 on Windows 11" or "PowerPoint Online on Chrome"
  2. Screenshot of the error — Windows: Win+Shift+S  |  Mac: Cmd+Shift+4
  3. Browser console log — press F12, click Console tab, screenshot any red errors
  4. Steps to reproduce — what did you click? Does it happen every time?

🆘 Quick Troubleshooting

Problem Solution
Popup blocked Allow popups for this site in browser settings
Sign-in button does nothing Check if popup blocker is enabled
Icons won't load Wait 30 seconds, refresh; check internet connection
Can't find Icon Library button Check Home tab; try closing and reopening Word/PowerPoint
"Missing bearer token" error Sign out and sign in again
Add-in won't install Make sure manifest.xml is not corrupted; try re-downloading
