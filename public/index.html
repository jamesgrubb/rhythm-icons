Quick Start Guide

👤 For IT Administrators Only: End users don't need to do anything - the add-in will appear automatically after you deploy it!

Deployment Steps (IT Admins):

  1. Download the manifest file above
  2. Sign in to Microsoft 365 Admin Center at admin.microsoft.com
  3. Go to Settings → Integrated apps → Add-ins
  4. Click "Deploy Add-in" and select "Upload custom apps"
  5. Upload the manifest.xml file you downloaded
  6. Choose who gets access (specific users/groups or everyone)
  7. Click Deploy - Done! The add-in will appear in Word and PowerPoint within 1-24 hours

For End Users:

Once deployed, users can find the add-in in:

  • Word/PowerPoint: Insert tab → Add-ins → Rhythm Icons
  • Click the icon button to open the taskpane
  • Sign in with Microsoft (first time only)
  • Browse, search, and insert icons!
